
# Tarot y Astrología Pro

Sistema profesional con:

- Next.js 14
- Login con Google (Auth.js)
- PostgreSQL + Prisma
- Tarot profesional
- Carta astral
- Panel admin
- Multiidioma

## Instalación

```bash
npm install
npm run dev
```

## Despliegue en Vercel

1. Subir a GitHub
2. Importar en Vercel
3. Configurar variables de entorno
4. Añadir dominio tarotastrologiapro.com
